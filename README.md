# Uolatility
Volatility 3.0 GUI

Uolatility는 최고의 메모리 포렌식 도구인 Volaility의 GUI를 도와주는 도구입니다.
이를 통해 조금 더 쉬운 분석을 하도록 도와줍니다.

사용을 하기 위해선 다음과 같은 환경 설정이 필요합니다.

정상적으로 Volatility 3.0이 돌아갈 수 있는 환경이 필요합니다.
JAVA가 정상적으로 설치되어 있어야합니다.
